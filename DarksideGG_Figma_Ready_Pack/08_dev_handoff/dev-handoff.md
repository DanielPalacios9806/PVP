# Handoff rápido para desarrollo

## Stack visual recomendado

- Frontend: Next.js + React + Tailwind CSS.
- Icons: Lucide, Heroicons o SVG propios.
- Animaciones: Framer Motion.
- Imágenes: WebP optimizado.

## Clases base sugeridas

- Background principal: `#05080C`.
- Cards: fondo `rgba(14,21,28,.78)`, borde `rgba(255,255,255,.08)`, radius `16px`.
- Botón principal: gradiente rojo `#FF2438 → #B90F24`.
- Botón secundario: fondo oscuro + borde cian.

## Reglas de implementación

- No exportar botones ni cards como imágenes.
- Usar SVG para logos e iconos.
- Usar WebP para hero y thumbnails.
- Mantener los textos reales en HTML para SEO.
- Mobile-first para navbar y bottom navigation.

# Darkside.gg — Figma Ready Pack

Este ZIP contiene una base organizada para reconstruir la landing de **Darkside.gg** en Figma sin depender de una sola imagen plana.

## Contenido principal

- `01_design_system/`: paleta, tipografía, tokens y reglas visuales.
- `02_assets/`: logos e íconos SVG base, mockups PNG/WebP y referencias.
- `03_figma_structure/`: cómo organizar páginas, frames y estilos en Figma.
- `04_components/`: inventario de componentes para crear con Auto Layout.
- `05_responsive/`: guía desktop/mobile y qué se oculta o resume en móvil.
- `06_copywriting/`: textos UI listos para pegar.
- `07_prompts/`: prompts para generar/refinar imágenes de hero, cards y banners.
- `08_dev_handoff/`: tokens CSS/JSON para pasar al frontend.

## Recomendación de uso

1. Importa los SVG en Figma.
2. Crea estilos de color con `tokens.darkside.json` o manualmente desde `design-system.md`.
3. Reconstruye botones, cards y navbars como componentes nativos de Figma.
4. Usa los PNG/WebP solo como referencia visual o como fondos temporales.
5. Redibuja o ajusta el isotipo en Illustrator/Figma si necesitas un logo final de producción.

## Nota importante

Los SVG incluidos son una **base vectorial editable** para trabajar rápido. Para marca final, conviene pulir el logo en Illustrator o Figma con curvas definitivas.

# Inventario de assets

## SVG

Usa SVG para logos, isotipos e iconos porque deben escalar sin perder calidad y poder cambiar de color en Figma.

## PNG/WebP

Usa PNG/WebP para mockups, fondos hero, banners e imágenes de torneos. En Figma puedes trabajar con PNG; para desarrollo web final conviene WebP.

## Qué NO exportar como imagen

Reconstruye estos elementos como componentes nativos de Figma:

- Botones
- Cards
- Navbar
- Bottom nav móvil
- Inputs
- Badges
- Stats
- Textos
- Tablas o grids

Esto permite editar responsive, estados y variantes sin rehacer imágenes.

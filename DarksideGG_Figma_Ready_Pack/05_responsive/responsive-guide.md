# Guía responsive

## Breakpoints

| Nombre | Ancho |
|---|---:|
| Mobile S | 360 px |
| Mobile M | 390 px |
| Mobile L | 430 px |
| Tablet | 768 px |
| Desktop | 1280 px |
| Desktop XL | 1440 px |

## Desktop landing

- Header horizontal completo.
- Hero en 2 columnas: texto + visual.
- Cards de juegos en 2 columnas.
- Torneos en grid/lista amplia.
- Stats, eventos y comunidad en 3 columnas.

## Mobile landing

La versión móvil debe priorizar conversión y lectura rápida:

- Header compacto con logo + Login + Registrarse + menú.
- Hero con visual reducido o integrado como fondo.
- CTA principal visible sin hacer mucho scroll.
- Stats en 3 columnas dentro de un solo card.
- Cards de juegos en 1 o 2 columnas según ancho.
- Solo 2 torneos destacados.
- Un widget final máximo.

## Ocultar en móvil

- Sponsor wall completo.
- Comunidad completa.
- Demasiadas tarjetas de torneos.
- Gráficos grandes.
- Tablas o brackets complejos.

## Medidas sugeridas mobile 390px

- Padding lateral: 20–24 px.
- Header height: 72–84 px.
- CTA height: 52–56 px.
- Card radius: 14–16 px.
- Espacio entre secciones: 24–32 px.
